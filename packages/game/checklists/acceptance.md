# 游戏验收测试清单

## 核心路径

- [ ] 游戏从启动到可玩状态加载完成（首屏 < 3s）
- [ ] 核心循环完整可执行：开始 → 游玩 → 胜负判定 → 重开
- [ ] UI 反馈正确：分数/生命/进度更新即时可见
- [ ] 胜利条件达成时触发正确表现（动画/音效/跳转）
- [ ] 失败条件达成时触发正确表现（Game Over/重试提示）

## 边界路径

- [ ] 首次启动（无存档状态）
- [ ] 连续快速点击/操作（防抖/节流）
- [ ] 极高分/极低分等极限数值显示
- [ ] 快速反复暂停/恢复（10 次以上）
- [ ] 游戏过程中切换浏览器标签页再返回
- [ ] 网络断开后重连（如果需要联网）
- [ ] 存储空间已满时存档失败的处理
- [ ] 所有按钮/交互区域在极端屏幕比例下的可点击性

## 兼容性

- [ ] Chrome 最新版
- [ ] Firefox 最新版
- [ ] Safari 最新版（特别注意 Web Audio / Canvas）
- [ ] iOS Safari（触摸 + 视口 + 音频 autoplay + 100vh 行为）
- [ ] Android Chrome（触摸 + 碎片化屏幕尺寸 + 导航栏重叠）

## 性能指标

| 指标 | 目标值（Canvas 2D） | 目标值（WebGL）| 测量方式 |
|------|-------------------|--------------|---------|
| 首屏加载 (FCP) | < 2s | < 2s | Lighthouse |
| 交互阻塞 (TBT) | < 50ms | < 50ms | Lighthouse |
| 帧率 (FPS) | >= 60（桌面）/ >= 30（移动）| >= 60（桌面）/ >= 30（移动）| DevTools Performance |
| 帧耗时 (Frame Time) | < 16ms | < 16ms | DevTools Performance |
| Draw Calls | — | < 100 | Spector.js / DevTools |
| 内存占用 | < 200MB | < 300MB | Chrome Task Manager |
| GPU 内存 | — | < 200MB | Chrome Task Manager |
| 资源总大小 | < 3MB | < 3MB | Network panel |
| 长时间运行（30min）| 无内存增长 > 50MB | 无内存增长 > 80MB | Heap snapshot 对比 |

## 无障碍

- [ ] 提供音效开关选项（癫痫警告）
- [ ] 动画可关闭（prefers-reduced-motion）
- [ ] 颜色对比度满足 WCAG AA（如果含文字）
- [ ] 键盘可完全操作（如果适用）
- [ ] **色盲模式**: 不使用仅靠颜色区分敌我/状态（同时使用图标/形状/文字）
- [ ] **色盲模式**: 提供高对比度方案或色盲滤镜

## 网络（如果需要联网）

- [ ] 弱网环境（Slow 3G）下核心功能可用
- [ ] 断网提示清晰，重连后自动恢复
- [ ] 排行榜/多人同步延迟 < 200ms

## WebGPU（如果使用）

- [ ] `navigator.gpu` 存在性检查，缺失时回退到 WebGL/Canvas 2D
- [ ] 设备丢失 (device lost) 事件处理：暂停渲染，尝试恢复
- [ ] 长时间运行后 GPU 内存无泄漏
