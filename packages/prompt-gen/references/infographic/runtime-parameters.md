# Runtime Parameters

信息图提示词生成的专属运行时参数。通用参数（`max_rounds`、`output_mode`、`language`、`mode`）的定义和提取规则见 `SKILL.md`。

## Aspect Ratio (宽高比)

| 值 | 描述 | 适用场景 |
|----|------|----------|
| `16:9` | 宽屏 | 演示文稿、视频、网页横幅 |
| `9:16` | 竖屏 | 手机屏幕、社交媒体故事 |
| `4:3` | 标准 | 传统演示文稿、打印 |
| `3:4` | 竖版标准 | 手机竖屏、打印 |
| `1:1` | 正方形 | 社交媒体帖子、图标 |
| `2:3` | 竖版照片 | 照片打印、海报 |
| `3:2` | 横版照片 | 照片打印、横幅 |
| `4:5` | 竖版社交 | Instagram 帖子 |
| `5:4` | 横版社交 | 传统照片 |
| `21:9` | 超宽屏 | 电影、超宽显示器 |
| `9:21` | 超窄竖屏 | 特殊展示 |

**默认值**：`16:9`

**推断规则**：
- 若用户明确指定宽高比/比例/aspect ratio，使用指定值
- 若用户提到"手机"、"竖屏"、"story"，推断为 `9:16`
- 若用户提到"演示"、"PPT"、"幻灯片"，推断为 `16:9`
- 若用户提到"社交媒体"、"Instagram"，推断为 `1:1` 或 `4:5`
- 否则使用默认值 `16:9`

## Image Size (图像尺寸)

| 值 | 描述 | 适用场景 |
|----|------|----------|
| `1k` | 1K 分辨率 (1024x576) | 快速预览、低配设备 |
| `2k` | 2K 分辨率 (2048x1152) | 标准质量，通用场景 |
| `4k` | 4K 分辨率 (4096x2304) | 高质量，打印/展示 |

**默认值**：`2k`

**推断规则**：
- 若用户明确指定 1k/2k/4k/分辨率/清晰度，使用指定值
- 若用户提到"打印"、"海报"、"展示"，推断为 `4k`
- 若用户提到"网页"、"屏幕"、"社交媒体"，推断为 `2k`
- 若用户提到"预览"、"快速"、"低配"，推断为 `1k`
- 否则使用默认值 `2k`

**注意**：`4k` 可能被某些图像生成模型拒绝，需处理错误并实施渐进式降级（4k → 2k → 1k）。

## Prompts Expand Mode (提示词扩写模式)

| 值 | 描述 |
|----|------|
| `auto` | 自动模式：评估提示词质量，不足则扩写 |
| `force` | 强制模式：始终扩写 |
| `disable` | 禁用模式：不扩写，直接使用原始提示词 |

**默认值**：`auto`

**推断规则**：
- 若用户明确指定 强制扩写/force expand，使用 `force`
- 若用户明确指定 不扩写/disable expansion，使用 `disable`
- 否则使用默认值 `auto`
