# 水平时间线布局 (Horizontal Timeline)

水平时间线布局定义。

## 描述

水平排列的时间线，适合线性时间序列展示。

## 特征

- **方向**：水平（从左到右）
- **元素**：时间点/里程碑沿水平线排列
- **连接**：箭头或线条连接各时间点
- **标签**：每个时间点下方或上方有标签
- **适合**：时间线、发展过程、历史事件

## 视觉元素

- 时间轴主线（水平线）
- 时间点标记（圆形/方形/菱形）
- 连接线/箭头
- 标签（标题 + 描述）
- 图标（可选）

## 适用内容类型

- `timeline`
- `process`（线性流程）

## 适用语气基调

- `professional`
- `educational`
- `technical`

## 示例

```
Horizontal timeline layout with 5 milestones from left to right, 
connected by arrows, each milestone has a circular marker and label below, 
clean modern design with corporate memphis style
```

## 变体

- `horizontal-timeline-arrow`：箭头连接
- `horizontal-timeline-line`：直线连接
- `horizontal-timeline-dashed`：虚线连接
