# 编号列表布局 (Numbered List)

编号列表布局定义。

## 描述

垂直或水平排列的编号列表，适合有序要点展示。

## 特征

- **方向**：垂直（默认）或水平
- **元素**：编号 + 标题 + 内容 + 图标
- **排列**：等间距垂直排列
- **适合**：列表、要点、步骤

## 视觉元素

- 编号（圆形/方形背景）
- 标题（粗体）
- 内容描述
- 图标（可选）
- 分隔线（可选）

## 适用内容类型

- `infographic-list`
- `process`（步骤列表）

## 适用语气基调

- `professional`
- `educational`
- `minimalist`

## 示例

```
Numbered list layout with 5 sections vertically arranged, 
each section has a numbered circle, title, description, and icon, 
clean modern design with corporate memphis style, 
equal spacing between sections
```

## 变体

- `numbered-list-vertical`：垂直排列（默认）
- `numbered-list-horizontal`：水平排列
- `numbered-list-card`：卡片式编号列表
