# 纵向分区布局 (Vertical Sections)

## 描述
内容按纵向划分为多个区块，从上到下依次呈现，适合展示流程、步骤或分类信息。

## 适用类型
- 流程说明（步骤 1 → 步骤 2 → 步骤 3）
- 分类展示（类别 A / 类别 B / 类别 C）
- 时间线叙事（过去 → 现在 → 未来）
- 层级信息（总览 → 详细 → 总结）

## 布局要点
- 区块数量：3-7 个（推荐 3-5 个）
- 排列方式：纵向堆叠，从上到下
- 区块间距：等间距或渐进式间距
- 宽度：全宽或居中固定宽度

## 视觉特征
- 清晰的视觉分隔（横线、留白、背景色）
- 每个区块有明确的标题或编号
- 阅读流：从上到下，单向顺序
- 可选：左侧图标或右侧图标

## 构图元素
- **标题区**：顶部主标题 + 可选副标题
- **区块结构**：
  - 编号/图标（可选，左侧或上方）
  - 区块标题（中号粗体）
  - 描述文字（小号常规）
  - 支持数据（可选，高亮显示）
- **连接元素**：箭头、虚线或留白分隔

## 示例场景
- "远程办公的 5 个优势"（5 个纵向区块）
- "产品开发流程"（需求 → 设计 → 开发 → 测试 → 发布）
- "健康饮食金字塔"（从底部到顶部的层级展示）
- "年度报告总结"（Q1/Q2/Q3/Q4 四个区块）

## 提示词关键词
vertical layout, stacked sections, top to bottom, sequential blocks, numbered list, step-by-step
