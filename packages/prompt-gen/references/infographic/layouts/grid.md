# 网格布局 (Grid Layout)

## 描述
内容按网格形式排列，每个单元格展示独立的信息点，适合展示并列的、平等重要的多个要点。

## 适用类型
- 特性对比（产品特性、服务优势）
- 数据展示（统计数据、关键指标）
- 选项列表（方案选择、功能模块）
- 要点总结（核心观点、关键发现）

## 布局要点
- 网格尺寸：2×2, 2×3, 3×3（推荐不超过 3×3）
  - **注意**：上述`X×X`是布局分析的元数据维度，不是提示词文本。写入提示词时应使用自然语言描述（如"两行三列的网格布局"），避免使用 `X×X 网格` 格式（违反 `shared-rules.md` 规则 9）。
- 单元格：等大小、等间距
- 对齐方式：网格对齐、居中排列
- 阅读顺序：从左到右、从上到下（Z 型）

## 视觉特征
- 高度规整、对称美感
- 每个单元格独立但视觉统一
- 明确的网格线或留白分隔
- 可选：卡片式设计、边框、背景色

## 构图元素
- **标题区**：顶部主标题
- **网格单元**：
  - 图标或数字（顶部或左上）
  - 单元标题（中号粗体）
  - 描述文字（小号常规，2-3 行）
  - 支持数据（可选，底部或右下）
- **视觉统一**：所有单元使用相同的布局模板

## 示例场景
- "产品 4 大特性"（2×2 网格）
- "6 个关键数据"（2×3 网格）
- "服务优势对比"（3×2 网格）
- "月度 KPI 总览"（3×3 网格）

## 提示词关键词
grid layout, matrix arrangement, equal cells, tiled design, card-based layout, organized grid
